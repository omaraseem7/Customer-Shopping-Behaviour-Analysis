# 📊 Customer Shopping Trends Analysis (Power BI + Excel)
This project analyzes customer shopping behavior to identify key trends in demographics, spending patterns, and product preferences. The goal is to generate insights that support data-driven business decisions.

## 🔍 Key Insights

- Customer demographics and segmentation analysis
- Revenue trends and spending behavior patterns
- Product category and payment method distribution
- Factors influencing customer spending and loyalty (Key Influencer analysis)

## 🛠 Tools Used

- Excel (Data Cleaning)
- Power BI (Dashboard & Visualization)

## 📈 Outcome

- An interactive Power BI dashboard that highlights customer behavior patterns and key drivers of revenue and loyalty.
